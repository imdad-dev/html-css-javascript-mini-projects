# Imsho.com — E‑commerce Front-end (Amazon-like clone)

A static front-end demo that mimics common Amazon-style shopping UI patterns using HTML, CSS and JavaScript.

## Quick analysis

- Similarities to Amazon front-end:
  - Header with logo, delivery location, category search, and cart/account controls.
  - Category navigation / "All" panel and top links like "Today's Deals".
  - Large hero slider/banner with promotional message and CTA.
  - Product grid sections grouped by category (Gaming, Fashion, Home, Books, Sports, etc.).
  - Product cards show image, title, price, original price, discount and an **Add to Cart** button.
  - Account/login menu, wishlist/orders links and a detailed footer with categories and help links.
 

This project is therefore a visual and interactive front-end approximation 

## Features (what's implemented)

- Responsive header with search and category selector
- Search recommendation dropdown (UI only)
- Login / Signup popup UI (client-side only)
- Hero slider and promotional banner
- Multiple category sections with product cards
- Add to Cart buttons (cart logic handled in `JavaScript/mainScript.js`)
- `cart.html` page to view cart (static / client-side)

## Project structure

- `index.html` — main product listing and landing UI
- `cart.html` — cart view
- `style.css` — styles for the site
- `JavaScript/mainScript.js` — cart and UI behaviours
- `image/` — product and asset images
