# ✨ Glow Profile Card

A modern and animated profile card built using **HTML, CSS, and JavaScript**. The project features a glowing profile picture, floating particles background, smooth hover effects, and social media icons, creating an attractive personal profile UI.

## 🚀 Features

* 🌟 Glowing animated profile image
* 🎨 Modern glassmorphism design
* ✨ Floating particle background animation
* 📱 Fully responsive layout
* 🔗 Social media icon section
* ⚡ Lightweight and beginner-friendly

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📂 Project Structure

```
Glow-Profile-Card/
│
├── index.html

├── profile.jpg
└── README.md
```

## 📸 Preview

The profile card displays:

* Profile image with glowing ring effect
* Name and profession/title
* Social media links
* Animated floating particles in the background

## 💡 Learning Outcomes

By building this project, you will learn:

* CSS animations
* Glassmorphism UI design
* DOM manipulation with JavaScript
* Dynamic particle generation
* Responsive web design
