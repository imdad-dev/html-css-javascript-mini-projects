# Zomato Landing Page Clone

A responsive landing page clone of Zomato, featuring a video background, navigation header, key business statistics, and app feature highlights. This project is built using basic front-end web technologies to replicate the modern look and feel of the food delivery platform.

---

## 🚀 Features

* **Video Background:** An immersive, muted, auto-playing video banner showcasing a premium brand feel.
* **Structured Header:** Fixed layout including the brand logo and a navigation menu (`Home`, `Investor Relations`, `Add Restaurants`, `Login`, `Contact Us`).
* **Dynamic Layouts:** Organized data sections using structured tables to display company milestones (restaurants, cities covered, and total orders delivered).
* **Feature Spotlights:** High-quality image integration highlighting specific application features.
* **Footer:** Standardized copyright protection.

---

## 🛠️ Tech Stack

* **HTML5:** Structured semantic markup (semantic sections, media embedding, lists, and tables).
* **CSS3:** Styling, positioning, custom fonts, navigation layouts, and responsiveness (handled via `style.css`).
* **JavaScript:** Added for future interactive components (e.g., handling navigation toggles, dynamic counter animations for stats, or authentication popups).
