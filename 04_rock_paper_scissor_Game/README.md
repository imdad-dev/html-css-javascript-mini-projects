# ✊✋✌️ Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. Players can choose Rock, Paper, or Scissors, and the computer will randomly generate its choice. The game keeps track of scores and displays the result of each round.

## 🎯 How to Play

1. Click on **Rock**, **Paper**, or **Scissors**.
2. The computer will automatically make a random choice.
3. The winner is decided based on the game rules:

| Player Choice | Computer Choice | Result |
| ------------- | --------------- | ------ |
| Rock          | Scissors        | Win    |
| Rock          | Paper           | Lose   |
| Paper         | Rock            | Win    |
| Paper         | Scissors        | Lose   |
| Scissors      | Paper           | Win    |
| Scissors      | Rock            | Lose   |

4. Scores are updated after every round.
5. Keep playing and challenge yourself!
