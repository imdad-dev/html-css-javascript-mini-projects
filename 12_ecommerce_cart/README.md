# Simple E-Commerce

A simple e-commerce web project built using **HTML, CSS, and JavaScript**. Users can view products, add items to the shopping cart, see the total price, and checkout.

## Features

- Display product list dynamically
- Add products to cart
- Show empty cart message
- Calculate total price
- Checkout button
- Simple and clean UI

## Technologies Used

- HTML
- CSS
- JavaScript
